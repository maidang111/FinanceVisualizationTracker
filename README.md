Key naming conventions:

Components are PascalCase.tsx — ChatPanel.tsx not chat-panel.tsx
Hooks are camelCase starting with use — useChat.ts
Services and utilities are camelCase — toolRenderer.ts
Routes/pages match the URL — Evals.tsx for /evals
Always have a .env.example — it shows professionalism and makes the repo usable by others
